# JavaScript Week 1 Practice Programs

## Overview

This repository contains basic JavaScript practice programs focused on fundamental programming concepts such as:

* Functions
* Arrays
* Objects
* Iteration
* Searching
* Dynamic Array Methods
* Comparison Operations

These programs are useful for beginners who are learning JavaScript basics and improving problem-solving skills.

---
