# Project Documentation

## Overview

This project is a web application developed using modern JavaScript technologies. It demonstrates structured application development with organized frontend and backend architecture.

The project is designed for learning and practicing:

* Full-stack web development
* Backend API creation
* Frontend integration
* Database operations
* RESTful services
* Modular coding practices

---

# Installation Guide

## Step 1: Extract the ZIP File

Extract the project into your desired folder.

---

# Backend Setup

## Step 2: Navigate to Backend Folder

```bash
cd backend
```

---

## Step 3: Install Backend Dependencies

```bash
npm install
```

---

## Step 4: Start Backend Server

```bash
npm start
```

Or:

```bash
node server.js
```

---

# Frontend Setup

## Step 5: Navigate to Frontend Folder

```bash
cd frontend
```

---

## Step 6: Install Frontend Dependencies

```bash
npm install
```

---

## Step 7: Run Frontend Application

```bash
npm run dev
```

Or:

```bash
npm start
```

---

# Environment Variables

Create a `.env` file inside the backend folder.

Example:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```
